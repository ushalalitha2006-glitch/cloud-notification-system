# ☁️ Cloud-Based Notification System

## 📌 Project Overview

The Cloud-Based Notification System is a serverless cloud application that allows users to send and track notifications in real time.

This project demonstrates core cloud computing concepts including:
- Event-driven architecture
- Serverless backend
- Cloud database integration
- Real-time email notifications
- Delivery tracking
- Cloud deployment

The system stores notifications in a cloud database and sends email alerts instantly using EmailJS.
